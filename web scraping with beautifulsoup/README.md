# Web Scraping Project - Quote & News Scraper

## Project Description
A comprehensive web scraping project that extracts quotes, authors, tags, and news headlines using BeautifulSoup. This project demonstrates multiple scraping techniques, interactive menus, and professional data handling.

## Files Included

### Main Scrapers
- `my_first_scraper.py` - Main quote scraper with detailed output and CSV export
- `scraper.py` - Advanced interactive menu scraper (quotes + news headlines)
- `simple_scraper.py` - Beginner-friendly version for learning basics
- `test_venv_complete.py` - Environment test script to verify installations

### Data Files
- `quotes_20260218_225548.csv` - 10 quotes from main scraper
- `scraped_data_20260218_222512.csv` - 20 items (10 quotes + 10 news headlines) from advanced scraper

### Documentation
- `requirements.txt` - Required Python packages
- `README.md` - Project documentation
